# Security policy

## Supported versions

Until the first stable release, security fixes are applied to the latest
published SeriesXRD version only.

## Reporting a vulnerability

Do not open a public issue for a suspected vulnerability. Email
`rmaples3@gatech.edu` with a description, reproduction steps, affected
versions, and any suggested mitigation. Do not include confidential
experimental data unless specifically requested through an agreed secure
channel.

You should receive an acknowledgment within seven days. A fix and disclosure
timeline will be coordinated after the report is reproduced and assessed.
